# 가오본갈비 홈페이지 프로젝트

## 프로젝트 개요

대전 동구 가오동에 위치한 갈비집 **가오본갈비**의 공식 홈페이지.
신규 손님 유치(검색 유입 → 방문 결정)를 목적으로 하는 싱글 페이지 반응형 사이트.

## 핵심 파일

| 파일 | 역할 |
|------|------|
| `js/config.js` | **나중에 수정할 항목 전부 여기에 있음** — 전화번호, 카카오/인스타 URL, 영업시간 |
| `data/menu.json` | 메뉴 데이터 — 가격 추가/수정은 여기서 |
| `assets/images/` | 음식·인테리어 사진 (블로그 스크린샷에서 크롭) |
| `blogContent/` | 블로그 스크린샷 원본 3장 (이미지 크롭 소스) |

## 식당 정보

- **전화:** 0507-1465-3581
- **주소:** 대전 동구 대전로448번길 52-4 1층 (가오동 590, 우편번호 34692)
- **영업시간:** 매일 11:30 ~ 22:00 (라스트오더 21:30)
- **브레이크타임:** 주말 15:00 ~ 17:00
- **대표 메뉴:** 수제 본갈비 1인분 16,000원

## 기술 스택

- 순수 HTML/CSS/JS 정적 사이트 (프레임워크 없음)
- Google Fonts: Noto Serif KR + Noto Sans KR
- 카카오맵 SDK (오시는 길 지도)
- Intersection Observer API (스크롤 페이드인)
- 배포: Vercel 또는 Netlify

## 페이지 섹션 구조

1. **Hero** — 풀스크린 갈비 사진 + 슬로건 + CTA 버튼
2. **Story** — 가게 소개, 고기 품질 강점
3. **Menu** — 카테고리 탭 (갈비류/추가메뉴/주류) + 카드 그리드
4. **Gallery** — 3열 이미지 그리드 + 인스타그램 링크
5. **Contact** — 전화 예약 + 카카오톡 채널 + 영업시간
6. **Map** — 카카오맵 임베드 + 주소 복사

## 나중에 추가할 항목

`js/config.js` 에서 수정:
- 카카오톡 채널 URL → `kakaoChannelUrl` 에 입력하면 버튼 자동 표시
- 인스타그램 URL → `instagramUrl` 에 입력하면 버튼 자동 표시
- 메뉴 가격 → `data/menu.json` 에서 `price` 값 입력

## 디자인 방향

- 화이트 모던 스타일 (THE SMASH 참고)
- 포인트 색상: 앰버/골드 (`#C8922A`) — 갈비 색감 연상
- 폰트: Noto Serif KR (제목) + Noto Sans KR (본문)
- 타겟: 가족 모임 + 외지 미식가 → "일부러 찾아올 만한 갈비집" 이미지
