## 1. 프로젝트 개요

### 목적
신규 손님 유치 — 검색 유입 시 강렬한 첫인상으로 방문 결정을 유도한다.

### 핵심 타겟
- 가족 모임을 계획하는 대전 인근 거주자
- 대전 밖에서 일부러 찾아오는 미식가

### 핵심 메시지
고기 품질과 공간 분위기 — "일부러 찾아올 만한 갈비집"

## 2. 기술 방향

| 항목 | 결정 |
|------|------|
| 구조 | 싱글 페이지 스크롤 (SPA) |
| 반응형 | 모바일 우선 (Mobile First) |

| SNS | 인스타그램 피드 임베드 |

## 3. 페이지 구조

### 섹션 1 — Hero
- **레이아웃:** 풀스크린 배경 이미지 (갈비 또는 인테리어 고퀄리티 사진)
- **콘텐츠:**
  - 로고 + 식당명 "가오본갈비"
  - 슬로건 (예: *"대전에서 가장 제대로 된 한 점"*)
  - CTA 버튼: `예약 문의하기` → 카카오톡 채널 or `tel:` 링크
- **네비게이션:** 고정(sticky) 상단 바 — 메뉴 / 스토리 / 갤러리 / 오시는길 앵커 링크
- **모바일:** 햄버거 메뉴로 네비게이션 접기

### 섹션 2 — 스토리 / 철학
- **레이아웃:** 좌우 분할 (텍스트 + 대표 이미지), 모바일에서는 세로 스택
- **콘텐츠:**
  - 가게 소개 한 문단
  - 고기 품질에 대한 자부심 (산지, 부위 선별 등)
  - 인테리어/공간 특징 한 줄

### 섹션 3 — 메뉴
- **레이아웃:** 카드 그리드 (PC 3열, 모바일 1열)
- **콘텐츠:**
  - 카테고리 탭: 갈비류 / 추가메뉴 / 주류
  - 각 카드: 사진 + 메뉴명 + 간단한 설명 + 가격
- **참고:** 메뉴 데이터는 별도 JSON 또는 정적 배열로 관리

### 섹션 4 — 갤러리 / SNS 피드
- **레이아웃:** 3열 이미지 그리드
- **콘텐츠:**
  - 인스타그램 최신 게시물 사진 (Instagram Basic Display API 또는 임베드)
  - "인스타그램 팔로우하기" 링크 버튼
- **폴백:** API 미사용 시 수동으로 선별한 정적 이미지 갤러리

### 섹션 5 — 예약 / 문의
- **레이아웃:** 중앙 정렬, 심플한 카드
- **콘텐츠:**
  - 카카오톡 채널 연결 버튼 (노란색 공식 버튼)
  - 전화번호 클릭 시 바로 전화 연결 (`tel:` 링크)
  - 영업시간 표기 (요일별)
  - 휴무일 표기

### 섹션 6 — 오시는 길
- **레이아웃:** 지도 임베드 (전체 너비) + 하단 텍스트 정보
- **콘텐츠:**
  - 카카오맵 or 네이버맵 iframe 임베드
  - 주소 (복사 가능하도록)
  - 주차 안내
  - 대중교통 안내 (선택)

### 푸터
- 식당명, 사업자 정보 (선택), 인스타그램 링크, 전화번호
- 저작권 표기
